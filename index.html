A Comprehensive Dataset for Wildfire Detection Using HPWREN Database

Fatemeh Taghvaei1,
1University of Illinois Chicago
Code Data

Abstract

We present a large-scale dataset for training neural networks for wildfire detection. We use videos from the HPWREN database and YouTube containing wildfires. We process each frame and divide each video into overlapping blocks of size 224x224. Each block is input to a neural network which was trained on another separate wildfire dataset. If the neural network detects a wildfire in the block, the block is saved. Then, each saved block is manually verified by a human to ensure the neural network hasn't misclassified the wildfire as something visually similar.
